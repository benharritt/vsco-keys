# VSCO Keys Keyboard Layout
My keyboard layout for [VSCO Keys] (https://github.com/vsco/keys).

### Exposure 

* **control + ↑** = +0.1
* **control + ↓** = -0.1
* **option + ↑** = +0.5 
* **option + ↓** = -0.5
* **command + ↑** = +1
* **command + ↓** = -1

### Basic Adjustments 

###### Highlights
* **shift + H** = +5
* **shift + G** = -5

###### Shadows
* **shift + S** = +5
* **shift + A** = -5

###### Whites
* **shift + W** = +5
* **shift + Q** = -5

###### Blacks
* **shift + B** = +5
* **shift + V** = -5
