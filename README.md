# VSCO Keys Keyboard Layout
My keyboard layout for [VSCO Keys] (https://github.com/vsco/keys).

### Exposure 

* **control + ↑** = +0.1
* **control + ↓** = -0.1
* **option + ↑** = +0.5 
* **option + ↓** = -0.5
* **command + ↑** = +1
* **command + ↓** = -1

### Basic

##### Shadow
* **S** = +5
* **A** = -5

##### Highlights
* **H** = +5
* **G** = -5

##### Whites
* **W** = +5
* **Q** = -5

##### Blacks
* **B** = +5
* **V** = -5
